# Frontend - Media Coverage Analysis

This is the React-based frontend for the Media Coverage project. It provides a dashboard for viewing news articles, AI-generated sentiment analysis, and media source management.

## 🚀 Tech Stack
* **Framework:** React (Vite)
* **Styling:** Tailwind CSS

## 🛠️ Features implemented (CP1)
* **Responsive Layout:** Dark/Light mode toggle with glassmorphism UI.
* **Navigation:** State-driven view switching (Dashboard, Analytics, Sources).
* **Mock Data Integration:** Ready-to-use components for news and analysis cards.

## 📦 Getting Started
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
